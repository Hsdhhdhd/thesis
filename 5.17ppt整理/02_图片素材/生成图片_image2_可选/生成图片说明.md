# image2 生成图片说明

这些图是可选补充素材，适合做 PPT 背景、动机页或流程页的视觉辅助。为了避免 AI 生成图片出现英文乱码，三张图都刻意要求无文字、无 logo、无水印。

文件用途建议：
- `image2_seated_lower_back_wearable_no_text.png`：标题页、系统概念页、haptic cue 说明页。
- `image2_closed_loop_pipeline_no_text.png`：sensing-to-cue pipeline / overall workflow 页。
- `image2_static_vs_movement_bout_no_text.png`：说明“同样坐 30 分钟，内部运动模式可能不同”的动机页。

生成提示词摘要：
- Seated lower-back wearable: anonymous seated adult, small sacrum/lower-back wearable, subtle sensing arcs and haptic pulse, professional academic style, no text.
- Closed-loop pipeline: wearable IMU, smartphone, laptop, signal waves, feature blocks, haptic cue return path, no text.
- Static vs movement bout: two side-by-side seated silhouettes, left static, right lower-trunk movement arcs and timeline markers, no text.
