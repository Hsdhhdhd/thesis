# Referenced Source Abstract Summaries

This folder covers the 47 sources currently cited in the thesis.

Important: the files contain non-verbatim abstract summaries, not copied publisher abstracts. This avoids copyright issues and keeps each item linked back to a verifiable DOI or official URL.

Recommended use:

- Use `all_abstract_summaries.md` for fast literature review and discussion writing.
- Use `index.md` to open one source at a time by citation key.
- Treat official hardware pages, datasheets, and product pages as specification evidence only, not as academic literature.
