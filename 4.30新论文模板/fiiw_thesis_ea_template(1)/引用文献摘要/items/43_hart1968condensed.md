## Non-verbatim Abstract Summary

This short classic paper introduces the condensed nearest-neighbor rule, which reduces the stored reference set used by nearest-neighbor classification. The central idea is to keep a smaller representative subset that can preserve the relevant decision behavior while lowering storage requirements.

## Relevance to This Thesis

This is the precise support for replacing a full KNN reference set with a smaller prototype/reference set before embedded deployment.
